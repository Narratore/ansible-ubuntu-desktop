desktop-slack
=============

Deploy slack on ubuntu desktop.

Requirements
------------

Ubuntu based system.

Role Variables
--------------

These are default variables for this role:

slack_version: 3.3.7

Dependencies
------------

Example Playbook
----------------

To use this role, just include it in your playbook, for example:

    - hosts: desktop
      roles:
         - desktop-slack
